Where do I add my own scripts?

Your .js files go in the /algos/ directory.

What information does the framework provide? What is the script expected to return?

The framework provides the maze dimensions, start and end position coordinates and an array with coordinates of the walls.

Your script should return an array path coordinate pairs [row, column], which will be drawn as the path in the maze.

There's a __stub.js file in the /algos/ folder. Copy it for your own script and use it as a guide for what your script accepts and returns.

How do I make my scripts appear in the pathfinding algorithms menu?

Currently, you have to add your script files in the index.html as an option for the select.algo-select.

Your options should look like <option value="filename.js">Algorithm name</option>

Don't add the file path, just the file name of your script in the /algos/ folder

I think my script is going to take a really long time to execute. Will this hang the tab?

Scripts are executed on a separate thread, using webworkers. Your tab should be fine :)

I'm getting a security error when I try initiating pathfinding.

You can't load web workers when running scripts from a local file. Open the page through a web-server (like the "Live Server" plugin for VS Code).
